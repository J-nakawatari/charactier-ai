# Archived Files

- システムログファイル（backend.log, frontend.log, dev.log など）
- 合計18個のログファイル

### `/debug-scripts/`

- 修正スクリプト（fix-*.js）

- キャラクター画像デザイン
- UI モックアップ（トップページ、ログイン画面、新規登録画面）

### `/docs-archive/`

- api-error-analysis-report.md
